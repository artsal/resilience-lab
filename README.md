# 🧪 Resilience Lab

An event-driven platform to simulate failure, recovery, and scalability in distributed systems.

## 🚀 Tech Stack
- Java, Spring Boot (Microservices)
- Apache Kafka, RabbitMQ
- MongoDB (NoSQL)
- Docker, Docker Compose
- Prometheus, Grafana
- Swagger (OpenAPI)

## 🎯 Goals
- Understand distributed system behavior
- Simulate failures and recovery
- Build observable systems

## 🧩 Current Status
- ✅ Event Service (Spring Boot + Swagger)
- ⏳ Kafka Integration (next)
